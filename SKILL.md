# COG253 Unified Skill Lattice

A single cohesive definition of 53 skills arranged on the topological skeleton
of Christopher Alexander's *A Pattern Language*. Each skill occupies a position
in the pattern lattice based on conceptual resonance — its operational scale,
structural role, and relationship to adjacent capabilities.

## How to Use This Document

1. **By pattern topology** — browse skills in their natural hierarchy (§1-§3)
2. **By name** — jump to any skill via the alphabetical index (§4)
3. **By domain cluster** — find related capabilities by theme (§5)
4. **By cross-reference** — each entry links to its source file and APL pattern
