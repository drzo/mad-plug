# hypergauge-orbifold

Provides a conceptual framework for understanding and designing neural architectures as Hypergauge Orbifolds. Use when analyzing the geometric properties of models, particularly in relation to symmetry, higher-order relationships, and state-space singularities.

This is a knowledge-based plugin that exposes skill documentation as guidance.
