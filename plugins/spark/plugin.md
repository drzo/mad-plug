# spark

Opinionated guidance for building modern web applications with a curated tech stack.
Provides structured stack variations matched to application complexity tiers (default
web app, content showcase, data dashboard, complex application) along with design
system, typography, color palette, component pattern, and performance optimization
reference material.

**Category:** Code Generation & Scaffolding  
**Version:** 0.1.0  
**Tags:** web, react, vite, typescript, scaffold, stack
