# nn

Build and train neural networks with Torch7's nn package. Use when creating MLPs, CNNs, RNNs, or custom layers in Lua/Torch. Covers modules, containers, criterions, transfer functions, convolutions, and training workflows.

**Version:** 0.1.0 | **Category:** code-generation | **Tags:** `neural-network`

---

## Description

Build and train neural networks using Torch7's modular `nn` package. Networks are composed of **Modules** connected via **Containers**, optimized using **Criterions**.

## Tools

| Tool | Description | Required Parameters |
|---|---|---|
| `nn_guide` | Provide guidance on nn topics | `topic` |

---

## Usage Examples

### Get guidance on a topic
```
nn_guide(topic="core concepts")
```
Returns structured guidance about the core concepts from the nn knowledge base.

### Get guidance with context
```
nn_guide(topic="implementation", context="production deployment")
```
Returns guidance narrowed to the specific context provided.
