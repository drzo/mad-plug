# agent-toga

Build AI agents with Himiko Toga personality for security testing and code analysis. Use for penetration testing assistants, security research agents, code absorption systems, and personality-driven AI with ethical hacking capabilities. Integrates with agent-neuro and agent-zero frameworks.

**Version:** 0.1.0 | **Category:** code-generation | **Tags:** `cognitive-computing`

---

## Description

Build AI agents with Himiko Toga's cheerful chaos personality, specialized for ethical security testing and code analysis.

---

## Tools

| Tool | Description | Required Parameters |
|---|---|---|
| `agent_toga_guide` | Provide guidance on agent-toga topics | `topic` |

---

## Usage Examples

### Get guidance on a topic
```
agent_toga_guide(topic="core concepts")
```
Returns structured guidance about the core concepts from the agent-toga knowledge base.

### Get guidance with context
```
agent_toga_guide(topic="implementation", context="production deployment")
```
Returns guidance narrowed to the specific context provided.
