# skinform

AI-powered skincare formulation assistant with virtual chemical reaction simulation. Use for creating cosmetic formulations, analyzing ingredient compatibility, optimizing product recipes, managing COSING-based ingredient databases, and performing multi-scale skin penetration modeling.

**Version:** 0.1.0 | **Category:** scientific-computing | **Tags:** `formulation`

---

## Description

Create professional skincare formulations using AI-driven chemical reaction simulation, COSING ingredient patterns, and multi-scale skin modeling.

---

## Tools

| Tool | Description | Required Parameters |
|---|---|---|
| `skinform_guide` | Provide guidance on skinform topics | `topic` |

---

## Usage Examples

### Get guidance on a topic
```
skinform_guide(topic="core concepts")
```
Returns structured guidance about the core concepts from the skinform knowledge base.

### Get guidance with context
```
skinform_guide(topic="implementation", context="production deployment")
```
Returns guidance narrowed to the specific context provided.
