# harmonic-llm

Build minimal LLM inference engines for the frequency domain. Use when creating harmonic oscillator chatbots, implementing the General Relevance architecture, or building models that operate directly on frequency-domain representations.

**Version:** 0.1.0 | **Category:** code-generation | **Tags:** `inference`

---

## Description

This skill provides tools to build minimal, efficient LLM inference engines that operate in the frequency domain, based on the principles of the General Relevance architecture.

## Usage Examples

### Get guidance on a topic
```
harmonic_llm_guide(topic="core concepts")
```
Returns structured guidance about the core concepts from the harmonic-llm knowledge base.

### Get guidance with context

Returns guidance narrowed to the specific context provided.
