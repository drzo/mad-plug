# Formulation Analyzer Plugin

Provides tools for analyzing and optimizing formulations across multiple domains:
skincare chemistry, neural network architectures, and mathematical structures.

## Capabilities

- **Skincare formulation** — Analyze compounds, look up ingredients, optimize concentrations
- **Neural topology** — Generate and visualize neural network architectures
- **Brain modeling** — Generate and visualize brain-inspired computational models
- **Mathematical tools** — Continued fractions and numerical analysis
- **TCN visualization** — Time crystal neuron model visualization

## Usage

Invoke when the user asks about:
- Formulation analysis ("analyze this skincare formulation")
- Ingredient research ("look up properties of niacinamide")
- Optimization ("optimize this formulation for stability")
- Neural architecture ("generate a topology for X")
- Visualization ("visualize this brain model")

## Domain Coverage

This plugin bridges the gap between chemical formulation science and
computational neuroscience, reflecting the repository's interdisciplinary nature.
