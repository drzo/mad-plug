# reservoirpy-nodes

Create custom ReservoirPy nodes for reservoir computing architectures. Use when implementing custom neural operators, trainable layers, online learning nodes, or parallel-trainable nodes for Echo State Networks and computational graph models.

**Version:** 0.1.0 | **Category:** pattern-recognition | **Tags:** `graph`

---

## Description

Create custom nodes for ReservoirPy's typed computational graph framework. ReservoirPy nodes are stateful operators that compose into arbitrary neural architectures.
