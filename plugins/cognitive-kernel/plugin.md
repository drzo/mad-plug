# cognitive-kernel

Build and operate cognitive development kernels based on Plan 9, Inferno-OS, and
OpenCog architectures. Provides tools for building kernel variants, configuring
distributed inference across compute nodes, managing compute clusters, and operating
cognitive service daemons (PIE-NN).

**Category:** Cognitive Computing  
**Version:** 0.1.0  
**Tags:** kernel, plan9, distributed, inference, opencog
