# pattern-navigator

Navigate and query the GH253 pattern topology. Provides three tools for exploring the
253 Christopher Alexander patterns mapped to GitHub/software domains: query by ID or
keyword, transform APL patterns to GitHub domain equivalents, and traverse the
broader/narrower relationship graph to discover connected design decisions.

**Category:** Pattern Recognition & Navigation  
**Version:** 0.1.0  
**Tags:** pattern, topology, navigation, graph
