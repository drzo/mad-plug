# language-creator (dsl-factory)

Design and create domain-specific languages end-to-end. Initialize language projects
with standard directory structure, define grammars using prime grammar specifications,
emit grammar files in PEG/BNF/ANTLR formats, extract terminology lexicons from text
corpora, and validate completed language projects for correctness.

**Category:** Language Design & DSLs  
**Version:** 0.1.0  
**Tags:** dsl, grammar, parser, language
