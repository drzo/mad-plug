# agent-zero

Build and extend Agent Zero autonomous AI agents. Use for creating custom tools, extensions, prompts, instruments, and agent profiles. Covers multi-agent cooperation, code execution, memory systems, MCP integration, and A2A protocol.

**Version:** 0.1.0 | **Category:** code-generation | **Tags:** `cognitive-computing`

---

## Description

Agent Zero is an organic, self-evolving agentic framework that uses the computer as a tool. Unlike predefined frameworks, it dynamically grows and learns through persistent memory and multi-agent cooperation.

## Usage Examples

### Get guidance on a topic
```
agent_zero_guide(topic="core concepts")
```
Returns structured guidance about the core concepts from the agent-zero knowledge base.

### Get guidance with context
```
agent_zero_guide(topic="implementation", context="production deployment")
```
Returns guidance narrowed to the specific context provided.
