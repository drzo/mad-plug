# multiplatform-api-weaver

Guide for integrating multiple platform APIs into unified CLI-like hybrid applications and SDKs. Use when building apps that orchestrate multiple services (Shopify, Stripe, QuickBooks, Slack, GitHub, etc.), creating SDKs that abstract platform differences, implementing advanced command combinations across APIs, or designing unified interfaces for heterogeneous backend systems.

**Version:** 0.1.0 | **Category:** code-generation | **Tags:** `cognitive-computing`

## Description

Build unified interfaces that weave multiple platform APIs into coherent CLI-like hybrid applications. This skill enables orchestration of heterogeneous services into a single, consistent developer experience.
