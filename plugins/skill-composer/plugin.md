# skill-composer

Decompose, transform, and compose skills across domains using structural analogy and
the General Relevance Transform (GRT). Extracts abstract templates from existing skills,
maps them to new domains through analogy, chains multiple transformations for multi-hop
domain transfer, and shares reusable skill scaffolds.

**Category:** Skill Engineering  
**Version:** 0.1.0  
**Tags:** skill, composition, transform, analogy
