# General Relevance Transform Skill

This skill provides a unifying framework for applying the General Relevance architecture to any domain of inquiry. It provides the tools and methods to transform traditional problems into the frequency domain and leverage the full power of the harmonic paradigm.

## Core Concepts

- **The General Relevance Transform (GRT)**: A mathematical operator that maps a problem from a time-domain or state-space representation to a frequency-domain representation in currence-sequence space.
- **Domain-Specific Prime Grammars**: The ability to define custom prime grammars that capture the fundamental structure of a specific domain.
- **Actionable Applications**: The transformation of theoretical concepts into practical, actionable applications for chatbot cognition.

## Quick Start

### 1. Define the Domain-Specific Prime Grammar

Use the `scripts/define_prime_grammar.py` script to define the prime factorization of the vocabulary for your domain.

### 2. Apply the General Relevance Transform

Use the `scripts/apply_grt.py` script to transform your problem data into the frequency domain.

### 3. Build and Run a Harmonic Model

Use the `harmonic-llm` and `harmonic-resonance-esn` skills to build and run a model on the transformed data.
