# mad-plug

A GitHub Copilot plugin repository with custom skills.

## Overview

This repository is configured as a Copilot coding agent plugin repository, enabling custom skills that extend Copilot's capabilities in your development workflow.

## Skills

Custom skills are defined in the `skills/` directory. Each skill is a YAML file describing an action Copilot can perform.

## Setup

This repository includes a [Copilot setup steps workflow](.github/workflows/copilot-setup-steps.yml) that configures the Copilot coding agent environment before it starts working.
