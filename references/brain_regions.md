# Time Crystal Brain Model - Regions Reference

This reference documents the hierarchical brain regions from Nanobrain Figure 7.15, extending the single neuron model to whole-brain architecture.

## Hierarchical Levels

The brain model spans 12 hierarchical levels from molecular to systemic scale:

| Level | Name | Scale | Key Components |
|-------|------|-------|----------------|
| 1 | Microtubule | Molecular | Protein structures, water channels, topology |
| 2 | Neuron | Cellular | Generalized neuron [3,4,3,3] |
| 3 | Cortical Branches | Columnar | Six-layer cortex, cortical columns |
| 4 | Cortex Domain | Regional | Lobes (A-D), Brodmann areas (E-H) |
| 5 | Cerebellum | Organ | Lobes (M,L,R), lobules (A-I) |
| 6 | Hypothalamus | Nuclear | Limbic nuclei, homeostatic centers |
| 7 | Hippocampus | Nuclear | Memory nuclei, spatial processing |
| 8 | Thalamic Body | Relay | Sensory relay nuclei |
| 9 | Skin Nerve Net | Peripheral | Cranial nerve branches |
| 10 | Cranial Nerve | Peripheral | 12 cranial nerves |
| 11 | Thoracic Nerve | Spinal | Motor/sensory rootlets |
| 12 | Blood Vessel | Vascular | Cerebral arteries |

## Notation Conventions

The brain model uses consistent notation patterns:

| Pattern | Meaning | Example |
|---------|---------|---------|
| Name(n) | n sub-elements | E(6) = 6 nuclei |
| Name[n] | n functional aspects | AIS[3] = 3 parts |
| Name* | Composite system | C*[4] = 4-part system |
| Name-Name | Compound structure | Ax-d = Axo-dendrite |
