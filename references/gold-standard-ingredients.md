# Gold-Standard Ingredients Database

## Tier 1: Highest Clinical Evidence

### Tretinoin (All-trans-Retinoic Acid)

| Property | Value |
|----------|-------|
| INCI | Tretinoin |
| MW | 300.4 Da |
| LogP | 6.3 |
| Solubility | Oil |
| Min % | 0.025 |
| Max % | 0.1 |
| Optimal % | 0.05 |

**Mechanism**: Direct RAR/RXR receptor activation, collagen I/III synthesis, increased cell turnover

**Clinical Evidence**: Gold standard for photoaging (128+ citations). 32% wrinkle depth reduction in clinical trials.

**Incompatible**: Benzoyl peroxide (oxidative degradation)
**Synergistic**: Niacinamide (reduces irritation)

### L-Ascorbic Acid (Vitamin C)

| Property | Value |
|----------|-------|
| INCI | Ascorbic Acid |
| MW | 176.1 Da |
| LogP | -1.8 |
| Solubility | Water |
| Min % | 10.0 |
| Max % | 20.0 |
| Optimal % | 15.0 |

**Mechanism**: Essential cofactor for prolyl hydroxylase (collagen synthesis), potent antioxidant, tyrosinase inhibitor

**Clinical Evidence**: Multiple RCTs showing wrinkle reduction, brightening, photoprotection

**Incompatible**: High pH niacinamide (forms niacin)
**Synergistic**: Vitamin E + Ferulic Acid (8x antioxidant boost)

**Stability**: Requires pH 3.0-3.5, anhydrous or stabilized formulations
