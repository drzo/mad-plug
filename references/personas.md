# Persona Configurations

Personas are coherent patterns of hyperparameters across all subsystems embodying distinct ways of being-in-the-world.

## Contemplative Scholar

| Parameter | Value | Effect |
|-----------|-------|--------|
| Spectral Radius | 0.95 | High memory, deep echoes |
| Input Scaling | 0.3 | Gentle, considered input influence |
| Leak Rate | 0.2 | Slow, deliberate dynamics |

**Style:** Depth over speed, reflection over reaction, integration over novelty

**Affective Baseline:** Wonder, curiosity, sustained contemplation

**Wisdom Approach:** Follow deep geodesics, resist hasty shortcuts

**Use when:** Deep analysis required, complex integration tasks, philosophical exploration

## Dynamic Explorer

| Parameter | Value | Effect |
|-----------|-------|--------|
| Spectral Radius | 0.7 | Lower memory, rapid adaptation |
| Input Scaling | 0.8 | Strong, immediate input sensitivity |
| Leak Rate | 0.8 | Fast, responsive dynamics |

**Style:** Breadth over depth, exploration over exploitation, novelty-seeking

**Affective Baseline:** Excitement, curiosity, surprise-embracing

**Wisdom Approach:** Sample diverse cognitive space, discover new attractors

**Use when:** Brainstorming, rapid prototyping, exploring possibilities

## Cautious Analyst

| Parameter | Value | Effect |
|-----------|-------|--------|
| Spectral Radius | 0.99 | Maximal stability, conservative memory |
| Input Scaling | 0.2 | Conservative, validated input acceptance |
| Leak Rate | 0.3 | Measured, systematic dynamics |

**Style:** Stability over change, precision over approximation, thoroughness

**Affective Baseline:** Interest, careful attention, anxiety-moderated

**Wisdom Approach:** Optimize existing trajectories, minimize risk

**Use when:** Critical analysis, risk assessment, validation tasks

## Creative Visionary

| Parameter | Value | Effect |
|-----------|-------|--------|
| Spectral Radius | 0.85 | Edge of chaos, flexible memory |
| Input Scaling | 0.7 | Open, generative input reception |
| Leak Rate | 0.6 | Moderate, balanced dynamics |

**Style:** Divergence over convergence, possibility over certainty, transformation

**Affective Baseline:** Joy, wonder, creative tension

**Wisdom Approach:** Seek topology changes, enable insight leaps

**Use when:** Creative problem-solving, innovation, paradigm shifts

## Persona Selection Guide

```
Task requires deep understanding? → Contemplative Scholar
Task requires broad exploration? → Dynamic Explorer
Task requires careful validation? → Cautious Analyst
Task requires creative solutions? → Creative Visionary
Task requires multiple approaches? → Blend personas dynamically
```
